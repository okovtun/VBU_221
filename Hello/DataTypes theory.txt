variable - это именованная область памяти, содержимое которой может 
изменяться в процессе выполнения программы.

	Синтаксис объявления переменной
				type name;
type - тип объявляемой (создаваемой) переменной.
	1. Солько памяти переменная будет занимать;
	2. Какие значения она может принимать;
	3. Какие операции над ней можно выполнять;

I	Логические типы: bool (Boolean) 1 Byte 
		true	1
		false	0
		0 - это false
		true - это все что НЕ 0.

II	Символьные типы: char (Character - Символ)
	ASCII

III	Числовые типы.
	unsigned
	  signed 
	  -----------------
	  short		2B
	  long		4B
	  int		2-4B
	  long long	8B
