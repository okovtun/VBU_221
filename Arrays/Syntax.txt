Array - это набор переменных одного типа в непрерывной области памяти.

			type name[SIZE];
type - тип элементов массива.
name - имя массива.
SIZE - количество элементов массива.
	   Может быть задано только 
	   целочисленным константным значением.

			const int SIZE = 5;
			int arr[SIZE];
